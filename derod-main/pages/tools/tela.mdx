## Key Features

| Feature | Description |
|---------|-------------|
| **On-Chain Storage** | Entire app (HTML/CSS/JS) stored in smart contracts |
| **Local Execution** | Runs in user's Engram wallet, not on servers |
| **Privacy-First** | User data encrypted, no tracking |
| **Censorship-Resistant** | Can't be taken down or blocked |
| **DERO Integration** | Native access to blockchain, wallets, tokens |

---

## TELA vs Traditional Web

| Aspect | Traditional Web | TELA |
|--------|----------------|------|
| **Hosting** | Centralized servers | DERO blockchain |
| **Storage** | Server databases | Smart contracts |
| **Execution** | Server-side | Local (Engram) |
| **Privacy** | Tracked, logged | Encrypted, private |
| **Censorship** | Can be blocked | Unstoppable |
| **Cost** | Monthly hosting fees | One-time blockchain fee |
