## How to Use TELA Links (Step-by-Step)

### **Step 1: Download & Launch Engram**
1. Download [Engram wallet](https://github.com/DEROFDN/Engram/releases) if you don't have it
2. Launch Engram on your computer
3. Create or restore your DERO wallet

### **Step 2: Enable Cyberdeck (XSWD Protocol)**
1. In Engram, go to **Settings**
2. Find **"Cyberdeck"** section
3. Click **"Turn On"** to enable XSWD (port 44326)
4. Keep Engram **running in the background**

### **Step 3: Click TELA Links**
1. With Engram running, click any **"Open"** button below
2. Engram will automatically catch the `tela://` link
3. Application opens in Engram

<Callout type="info">
  **First Time?** Engram will ask permission to connect. Click "Allow" to enable TELA link handling.
</Callout>

**Troubleshooting:**
- ‚ùå **"TELA handler not ready"** ‚Üí Engram is not running OR Cyberdeck is not enabled
- ‚ùå **Nothing happens** ‚Üí Check Engram is set as default `tela://` protocol handler
- ‚úÖ **App opens** ‚Üí Success! Cyberdeck + Engram properly configured

**Cyberdeck Status:**
- üü¢ **Green** = Cyberdeck enabled, XSWD active on port 44326
- ‚ö´ **Gray** = Cyberdeck disabled, TELA links won't work
